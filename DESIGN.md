# AI-Native Bootcamp — дизайн-код

> Извлечено напрямую из CSS в `src/pages/index.astro` (инлайн `<style is:global>`).
> Внешних CSS-файлов нет (кроме самохостируемых шрифтов `@fontsource`).
> Стиль: «терминальный брутализм» — прямые углы, монопространство, один тёплый акцент.

---

## 1. Базовые принципы

| Принцип | Реализация |
|---|---|
| **Нулевые скругления** | `*, *::before, *::after { border-radius: 0 !important; }` — углы прямые везде (исключения: круглые точки-индикаторы и аватары через `50% !important`) |
| **Две гарнитуры, и только две** | Roboto Condensed (заголовки) + JetBrains Mono (весь текст/UI) |
| **Один акцент** | `--accent` (#f5a623) — единственный цвет вне чёрно-кремовой пары |
| **Сетки на бордерах** | Контентные блоки — это `grid` с `border: 1px solid var(--ink10)` и внутренними разделителями, без gap |
| **Контейнеры** | `body` max-width **1440px**, контентные сетки — **1200px**, текстовые блоки — **640–720px** |
| **Табличные цифры** | `font-variant-numeric: tabular-nums` на `body` и заголовках |
| **Базовый кегль** | `html { font-size: 16px; }`, плавный скролл |

## 2. Цветовая палитра

Объявлена в `:root` ([index.astro:38](src/pages/index.astro#L38)):

| Токен | Значение | Назначение |
|---|---|---|
| `--bg` | `#f5f5f0` | Фон — светло-кремовый (off-white) |
| `--ink` | `#0a0a0f` | Основной — почти чёрный с синевой |
| `--accent` | `#f5a623` | Акцент — тёплый оранжевый/янтарный |

### Прозрачностная шкала «ink»

Вместо отдельных серых — один чёрный с разной альфой. Это и есть основа цветового ритма.

| Токен | Альфа | Типичное применение |
|---|---|---|
| `--ink05` | 0.05 | Фон карточек (`.role-card`, `.stat-box`, `.feedback-card`) |
| `--ink10` | 0.10 | Бордеры сеток, разделители секций |
| `--ink20` | 0.20 | Слабые бордеры, «призрачные» цифры (`.lesson-num-sm`) |
| `--ink40` | 0.40 | Третичный текст, метки, мета |
| `--ink60` | 0.60 | Вторичный текст (`.hero-sub`, `.about-text`, описания) |

### Инверсия в тёмных блоках

В `.cta-block` (тёмная секция тарифов) палитра переворачивается:
- фон `--ink`, текст `--bg`;
- бордеры/приглушённый текст не через `--inkXX`, а через `rgba(245,245,240, 0.12 / 0.45)` (кремовый с альфой);
- акцент `--accent` сохраняется как единственный цветной элемент.

## 3. Цветовые акценты

Оранжевый `--accent` используется строго дозированно — как «маркер внимания», не как заливка:

| Где | Как |
|---|---|
| Заголовки | Слово-акцент `.hero-accent` внутри H1/H2 (`color: var(--accent)`) |
| Метки секций | `.bracket-label`, `.hero-label` — целиком оранжевые |
| Цифры-статы | `.stat-num` (48px) — крупные оранжевые числа |
| Левые рамки | `border-left: 2px solid var(--accent)` у `.hero-sub`, `.feedback-stack`, `.lesson-concepts` |
| Технические подписи | `.committee-role`, `.lesson-concepts`, `.feedback-stack` — стек/роль оранжевым моноширинным |
| Кнопки | Заливка `.cta-btn` / `.nav-cta` в покое |
| Ховеры | Логотип, ссылки навигации, стрелки карусели → оранжевый |
| Фокус | `outline: 2px solid var(--accent)` |
| Пульс | `@keyframes` индикатора набора — пульсирующая тень `rgba(245,166,35,...)` |

**Правило:** на любом экране акцент занимает малую площадь — пара слов, цифра, рамка или одна кнопка.

## 4. Цветовой ритм

Ритм построен не на множестве цветов, а на **чередовании плотности чёрного**:

```
фон (--bg кремовый)
  └─ карточка (--ink05, едва темнее фона)
       └─ бордер (--ink10)
            └─ текст вторичный (--ink60)
                 └─ текст основной (--ink почти чёрный)
                      └─ ⚡ акцент (--accent) — единичные вкрапления
```

Секционный ритм по вертикали:
- каждая `section` разделена `border-bottom: 1px solid var(--ink10)` — тонкие линии задают «техническую сетку» страницы;
- ровно одна секция (`.cta-block`, тарифы) — **тёмная инверсия**, создаёт визуальную паузу/кульминацию в середине лендинга;
- внутри секций — повторяющийся паттерн `[bracket-label оранжевый] → [section-title чёрный с акцентным словом] → [контент-сетка на бордерах]`.

## 5. Иерархия шрифтов

### Гарнитуры

| Роль | Шрифт | Начертания (подключены) |
|---|---|---|
| **Дисплейная** (заголовки, цифры) | **Roboto Condensed** | 400 / 700 / 900 (latin + cyrillic) |
| **Текстовая / UI** (всё остальное) | **JetBrains Mono** | 400 / 500 / 600 (cyrillic) |

Фолбэки: Roboto Condensed → `'Arial Narrow', Arial, sans-serif`; JetBrains Mono → `'SF Mono', 'Consolas', monospace`.
`body` по умолчанию — JetBrains Mono; `h1–h3, .display` принудительно (`!important`) — Roboto Condensed 900 uppercase.

### Характер набора

| Параметр | Заголовки (Roboto Condensed) | Текст (JetBrains Mono) |
|---|---|---|
| Регистр | UPPERCASE | смешанный; метки — UPPERCASE |
| Letter-spacing | отрицательный (`-0.01…-0.03em`) — плотный | положительный (`0.04…0.15em`) у меток |
| Line-height | 0.92–0.95 (тугой) | 1.3–1.8 (воздушный) |
| Вес | 900 (H1/H2), 700 (модули) | 400 / 600 |

## 7. Кнопки

| Класс | В покое | Ховер | Применение |
|---|---|---|---|
| `.cta-btn` | заливка `--accent`, текст `--ink`, mono 12px uppercase, ls 0.12em, padding `16×40` | чёрная шторка `::before` въезжает слева (`translateX -100%→0`), текст → кремовый | Главный CTA (запись/виджет) |
| `.nav-cta` | то же, padding `0×28`, на всю высоту нав-бара | та же slide-in шторка | CTA в навигации |
| `.btn-invite` | заливка `--ink`, текст `--bg`, бордер `--ink`, mono 600 | инверсия в акцент | Вторичное действие (видео-модалка) |
| `.format-floor__btn` | заливка `--ink`, текст `--bg`, Roboto Cond. 900 uppercase | фон → `--accent`, текст → `--ink` | Кнопки в модалках/хинтах |

Общий приём ховера — **slide-in заливка** через псевдоэлемент `::before` с `transform: translateX`, а не смена `background` (даёт «механический» переход в духе терминала). Длительность `0.3s ease`.

## 8. Прочие элементы

### Карточки и сетки
- Базовый паттерн: контейнер `border: 1px solid var(--ink10)`, дети — ячейки с `border-right/bottom: 1px solid var(--ink10)`, последняя в ряду обнуляет бордер. `gap: 0`.
- Фон ячеек — `--ink05` (`.role-card`, `.stat-box`, `.feedback-card`).
- Сетки: hero-статы `repeat(3,1fr)`, аудитория/роли `2×`, авторы `repeat(4,1fr)`.

### Тени (единственное место)
- `.format-modal__dialog` — «жёсткая» офсет-тень `box-shadow: 12px 12px 0 var(--ink)` (на моб. `6px 6px 0`). Без размытия — в стиле необрутализма. Везде остальное — без теней.

### Анимации
| Эффект | Механика |
|---|---|
| Появление блоков | `.reveal` → `.visible` через `IntersectionObserver`; `opacity 0→1` + `translateY(30px→0)`, `0.8s ease-out`. Активно только при классе `.js` на `<html>` (без JS контент виден сразу) |
| Слова hero | `.hero-title .word` всплывают по очереди (`translateY(20px)`, `0.5s`) |
| Счётчики | `.stat-num[data-target]` — JS-инкремент при появлении в вьюпорте |
| Индикатор набора | пульсирующая `box-shadow` оранжевым |
| Карусель отзывов | бесконечный луп (клоны + телепорт), на тач — `touch-action: pan-y` и кламп жеста в одну карточку |

### Брейкпоинты
`992px` (включается бургер-меню) · `900` · `768` · `640` · `600` · `560` · `480`.
Основные переломы — **900 / 768 / 640 / 480**. На мобилке секции: `padding: 48px 16px` (десктоп — `80px 24px`), многоколоночные сетки схлопываются в 1 колонку.

### Фокус и доступность
- `:focus-visible { outline: 2px solid var(--accent); outline-offset: 2px; }`.
- `-webkit-tap-highlight-color: transparent` на интерактиве.
- Клоны карусели помечены `aria-hidden="true"`.

## 9. Шпаргалка значений

```css
/* Палитра */
--bg:#f5f5f0; --ink:#0a0a0f; --accent:#f5a623;
--ink05/10/20/40/60: rgba(10,10,15, .05/.10/.20/.40/.60);

/* Шрифты */
Заголовки: 'Roboto Condensed' 900, uppercase, ls -0.03em, lh 0.95
Текст/UI:  'JetBrains Mono' 400/600, ls +0.04…0.15em (метки)

/* Контейнеры */
body 1440 · сетки 1200 · текст 640–720

/* Радиусы / тени */
border-radius: 0 (всюду)
shadow: только модалка — 12px 12px 0 var(--ink)

/* Переходы */
ховер-кнопки: slide-in ::before, transform translateX, 0.3s ease
reveal: opacity+translateY, 0.8s ease-out
```
